MSTY Project 완전판 사용방법

1. 휴대폰에서 바로 확인
- ZIP 파일을 다운로드합니다.
- 삼성 '내 파일' 앱 → 다운로드 → ZIP 파일 선택 → 압축 풀기.
- 압축을 푼 폴더 안의 index.html을 누릅니다.
- Chrome 또는 삼성 인터넷으로 엽니다.

2. 이번 버전이 흰 화면으로 깨지지 않는 이유
- 디자인(CSS)과 계산(JavaScript)이 전부 index.html 하나 안에 들어 있습니다.
- styles.css나 app.js를 따로 찾지 않으므로 index.html만 열어도 정상 표시됩니다.
- manifest.webmanifest와 icon.svg는 나중에 웹주소로 배포할 때 앱 아이콘용입니다. 없어도 화면과 기능은 정상 작동합니다.

3. 처음 들어 있는 실제 기준값
- 보유 주수: 215주
- 평가금: $2,728.18
- 평가손익: -$1,255.11 (-31.5%)
- 투입 원금: $3,983.29
- 평균 단가: 약 $18.53
- 2026년 6월 배당: $91.82
- 2026년 7월 배당: $101.14
- 입력된 누적 세후 배당: $192.96

4. 데이터 저장
- 거래, 배당, 설정은 현재 브라우저의 내부 저장소에 자동 저장됩니다.
- 브라우저 데이터 삭제 또는 다른 기기로 이동하기 전에는 설정 → 데이터 백업(JSON)을 사용하세요.
- 새 기기에서는 설정 → 백업 불러오기로 복원할 수 있습니다.

5. 탭별 기능
- 홈: 평가금, 평가손익, 평단, 주수 목표, 배당, 회수율, 금액 표시 그래프.
- 거래: 매수/매도/재투자 입력, 평단 및 평가손익 자동 계산.
- 배당: 세후 배당 입력/수정/삭제, 월별 합계와 금액 표시 그래프.
- 목표: 1,000주 로드맵, 남은 매수금, 최근 4회 기준 예상 배당.
- 설정: 보유 스냅샷, 현재가, 목표값, 백업/복원.

6. 무료 웹주소 배포
- 화면과 수치를 확인한 뒤 이 폴더 전체를 Cloudflare Pages 또는 GitHub Pages에 올리면 됩니다.
- 배포 전에도 index.html 파일 자체로 모든 기능을 확인할 수 있습니다.
